## O que é o CodeForge

**CodeForge** é um editor de código rápido, leve e flexível.
Combina simplicidade com recursos avançados para todo o ciclo de desenvolvimento:

* edição inteligente

Atualizações frequentes trazem novas funcionalidades e correções.

## Extensões Incluídas

O **CodeForge** inclui extensões embutidas para várias linguagens.

Extensões com suporte completo usam o sufixo:

<language>-language-features

Exemplo:

* `json` → syntax highlight
* `json-language-features` → IntelliSense, Go to Definition, etc.
